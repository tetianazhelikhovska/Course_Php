ПРОЕКТ: ПОГОДНЕ ЗАСТОСУВАННЯ

АНОТАЦІЯ

Це веб-додаток для відображення інформації про погоду в обраному місті. Проект розроблений на PHP з використанням принципів об'єктно-орієнтованого програмування та паттерну Strategy через інтерфейси.

Додаток надає зручний інтерфейс для перегляду поточної температури та умов погоди з відповідним динамічним оформленням сторінки залежно від типу погоди.


ФУНКЦІОНАЛ

Отримання даних про погоду:
Додаток використовує класи для отримання інформації про погоду за назвою міста. Реалізовано три варіанти джерел даних: отримання даних з віддаленого сервера, генерація випадкових даних та використання тестових даних.

Конвертація температури:
Дані про температуру зберігаються в Кельвінах. Додаток автоматично конвертує температуру в градуси Цельсія та Фаренгейта для зручності користувача.

Динамічне оформлення:
На основі типу погоди (сонячна, хмарна, снігова чи штормова) автоматично змінюється фоновий образ та підбирається відповідна велика ікона погоди.

Відображення інформації:
Сторінка показує місто, поточну дату та день тижня, температуру та ймовірну погоду. Весь контент правильно екранується для безпеки.

Адаптивний дизайн:
Додаток має сучасний адаптивний дизайн з використанням CSS і SVG ікон, що забезпечує правильне відображення на різних розмірах екранів.

Архітектура з використанням інтерфейсів:
Код побудований таким чином, що легко можна замінити джерело даних про погоду без змін основної логіки додатку.

                          ПРОЕКТ: IMAGE GALLERY
================================================================================

НАЗВА ПРОЕКТУ:
Image Gallery - Фотогалерея для відображення та перегляду колекції зображень

================================================================================
АНОТАЦІЯ:
================================================================================

Це веб-застосунок на PHP, розроблений для створення інтерактивної фотогалереї з 
можливістю переглядання колекції зображень. Проект демонструє основні концепції 
веб-програмування на PHP, включаючи роботу з GET-параметрами, масивами даних та 
обробку URL-кодування для безпеки. Галерея дозволяє користувачам переглядати 
мініатюри всіх доступних зображень та отримувати детальну інформацію про кожне 
зображення, включаючи його назву та розширений опис.

================================================================================
ОПИСАННЯ ФУНКЦІЙ ПО АБЗАЦАХ:
================================================================================

ОСНОВНА ФУНКЦІЯ - УПРАВЛІННЯ ГАЛЕРЕЄЮ:
Проект містить централізовану систему управління зображеннями через файл 
images.inc.php, де зберігаються все дані про фотографії. Кожне зображення має 
унікальний ідентифікатор (назву файлу), назву та розширений опис. Дані 
організовані у вигляді асоціативних масивів PHP, що дозволяє легко додавати нові 
фотографії до колекції без змінення логіки застосунку.


ГОЛОВНА СТОРІНКА ГАЛЕРЕЇ:
Сторінка gallery.php відповідає за відображення всіх доступних зображень у 
вигляді сітки. Вона перебирає масив $imageTitles та створює для кожного 
зображення картку з мініатюрою та назвою. Кожна картка оформлена як гіперпосилання, 
яке передає параметр image через GET-запит, дозволяючи користувачу вибрати 
зображення для детального перегляду. Сторінка використовує функції безпеки для 
правильного кодування назв файлів та текстового вмісту.


СТОРІНКА ДЕТАЛЬНОГО ПЕРЕГЛЯДУ ЗОБРАЖЕННЯ:
Файл image.php обробляє запити про конкретне зображення та відображає його у 
повному розмірі з детальною інформацією. Сторінка перевіряє валідність 
GET-параметра 'image' та повертає відповідне зображення та його опис. Якщо 
користувач спробує отримати доступ до неіснуючого зображення, виводиться 
повідомлення про помилку. Описи зображень динамічно форматуються - розриви рядків 
в базі даних конвертуються в HTML-теги <br /> для правильного відображення.


ФУНКЦІЯ БЕЗПЕКИ ТА САНІТИЗАЦІЇ:
Проект використовує допоміжну функцію e(), визначену у functions.inc.php, для 
захисту від XSS-атак. Ця функція кодує усі спеціальні символи HTML за допомогою 
htmlspecialchars(), перетворюючи потенційно шкідливий вміст у безпечний текстовий 
формат. Крім того, для роботи з назвами файлів використовуються функції 
rawurlencode() та rawurldecode(), що гарантує правильну передачу спеціальних 
символів через URL-рядок.


МОДУЛЬНА АРХІТЕКТУРА:
Проект організований за принципом розділення відповідальності - мова-специфічна 
логіка розташована в файлах inc/, HTML-шаблони в папці views/, а CSS-стилі в 
папці styles/. Така структура робить код легким для обслуговування та масштабування. 
Компоненти header.php та footer.php використовуються на всіх сторінках проекту, 
забезпечуючи єдиний дизайн та уникаючи дублювання коду.


ВІЗУАЛЬНЕ ОФОРМЛЕННЯ ТА СТИЛІЗАЦІЯ:
Проект включає кілька CSS-таблиць стилів (custom.css та simple.css) для 
оформлення інтерфейсу користувача. Галерея розташована в контейнері 
'gallery-container', а кожне зображення представлено як окремий елемент 
'gallery-item'. Стилі дозволяють створити адаптивний дизайн, що розумно 
поводиться на різних розмірах екранів, та забезпечують приємний візуальний 
досвід при перегляді фотографій.


КОЛЕКЦІЯ ЗОБРАЖЕНЬ:
Галерея містить шість унікальних пейзажних фотографій: яскравий захід сонця 
в каньйоні, барвисті пустельні дюни, величні засніжені гірські вершини, 
спокійна річка з замком на схилі, осінній гірський пейзаж та мальовничий 
пляжний будинок. Кожна фотографія супроводжується детальним описом, який 
розповідає про атмосферу, композицію та особливості кожної сцени.

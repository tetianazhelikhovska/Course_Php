ОПИС ПРОЕКТУ
────────────────────────────────────────────────────────────────────────────────

Це веб-додаток на PHP, розроблений для перегляду та управління інформацією про 
міста по всьому світу. Додаток надає користувачам інтерактивний інтерфейс для 
навігації по базі даних світових міст, перегляду детальної інформації про кожне 
місто та виконання операцій редагування. Проект демонструє сучасні практики 
розробки на PHP, включаючи об'єктно-орієнтоване програмування та шаблон Repository.


ФУНКЦІОНАЛЬНІСТЬ
────────────────────────────────────────────────────────────────────────────────

Перегляд міст та пагінація
Додаток відображає міста списком з пагінацією, по 15 міст на сторінці. Користувачі 
можуть навігувати базою даних за допомогою кнопок "Попередня" та "Наступна". 
Система пагінації автоматично розраховує загальну кількість міст і коригує 
навігаційні елементи управління.

Інформація про міста
Кожне місто відображається з його назвою, країною та емодзі флагом, який представляє 
країну походження. Додаток генерує Unicode флаги на основі кодів ISO 3166-1 alpha-2, 
забезпечуючи легке розпізнавання кожної країни та поліпшуючи користувацький інтерфейс.

Детальна інформація про місто
Користувачі можуть клацнути на місто, щоб переглянути детальну інформацію, включаючи 
географічні координати, статистику населення, назву в ASCII форматі, інформацію про 
адміністративний район та ISO коди країн.

Редагування даних міст
Додаток дозволяє редагувати інформацію про міста в базі даних. Користувачі можуть 
змінювати назву міста, країну, ISO код країни та кількість населення. Усі зміни 
зберігаються в базі даних.

Інтеграція з базою даних
Додаток використовує реляційну базу даних (MySQL/MariaDB) для зберігання інформації 
про міста. Застосовуються підготовлені запити та параметризовані значення для 
безпеки та запобіганню SQL-ін'єкцій.


СТРУКТУРА ПРОЕКТУ
────────────────────────────────────────────────────────────────────────────────

index.php               - головна сторінка зі списком міст
city.php                - детальна інформація про місто
edit.php                - редагування міста

inc/
  all.inc.php           - підключення необхідних файлів
  db-connect.inc.php    - налаштування підключення до БД
  functions.inc.php     - допоміжні функції

src/
  WorldCityModel.php    - модель даних міста
  WorldCityRepository.php - робота з базою даних

views/
  layouts/
    main.view.php       - головний шаблон
  pages/
    index.view.php      - шаблон списку міст
    city.view.php       - шаблон детальної інформації
    edit.view.php       - шаблон редагування

styles/
  custom.css            - стилі
  simple.css            - базові стилі


ВИКОРИСТАННЯ
────────────────────────────────────────────────────────────────────────────────

1. Откройте index.php, щоб переглянути список міст
2. Клацніть на місто для перегляду детальної інформації
3. Скористайтесь кнопкою редагування для змін
4. Використовуйте навігацію для перегляду інших міст

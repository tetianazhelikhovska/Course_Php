АНОТАЦІЯ ДО ПРОЕКТУ "NAME EXPLORER"

Назва проекту: Name Explorer (Дослідник імен)

Опис проекту:
Проект представляє собою веб-додаток для дослідження та пошуку особистих імен. Програма дозволяє користувачам переглядати статистику найпопулярніших імен, досліджувати імена за першою літерою алфавіту та отримувати детальну інформацію про конкретне ім'я.

Основний функціонал:

1. Головна сторінка (індексна сторінка)
Головна сторінка додатку виступає вхідною точкою для користувача. На цій сторінці відображається топ-10 найпопулярніших імен за загальною кількістю реєстрацій. Ця функція допомагає користувачам миттєво бачити тенденції та найбільш вживані імена у базі даних.

2. Навігація за алфавітом
Додаток забезпечує интуїтивну навігацію через пайку с 26 букв англійського алфавіту, розташовану в заголовку сторінки. Кожна літера є посиланням, за натисканням якого користувач переходить до сторінки з переліком усіх імен, що починаються на цю букву. Активна літера виділяється спеціальним стилем, щоб користувач завжди знав, на якій букві він знаходиться.

3. Перегляд імен за початковою літерою
Сторінка дисплея імен за буквою показує всі імена, що починаються на обрану букву, відсортовані в алфавітному порядку. Імена виводяться з паджинацією - по 15 імен на сторінку, що забезпечує зручний перегляд великого обсягу даних. Користувач може переміщатися між сторінками паджинації, щоб побачити всі доступні імена на цю букву.

4. Детальна інформація про ім'я
При натисканні на конкретне ім'я користувач переходить на сторінку детальної інформації про це ім'я. На цій сторінці виводяться всі історичні записи для даного імена, включаючи рік реєстрації та кількість використань цього імена протягом кожного року. Дані відсортовані в хронологічному порядку від найдавніших записів до найновіших.

Архітектура проекту:

Проект розроблений з використанням архітектурного паттерну MVC (Model-View-Controller). Він складається з трьох головних модулів:

- Модель (inc/names.inc.php): Містить функції для роботи з базою даних, включаючи запити для отримання імен за першою букву, пошуку даних про конкретне ім'я та генерування статистики.

- Вид (views/pages/): Включає три файли представлення - для головної сторінки, сторінки з переліком імен та сторінки детальної інформації про ім'я.

- Контролер (index.php, char.php, name.php): Три файли контролерів, які обробляють запити користувача, взаємодіють з моделлю та передають дані до представлення.

Технологічний стек:

Проект розроблений на мові PHP з використанням бази даних MySQL. За допомогою підготованих SQL-запитів (prepared statements) забезпечується захист від SQL-ін'єкцій. HTML та CSS використовуються для структури та стилізації користувацького інтерфейсу. Додаток підтримує кодування UTF-8 для коректної роботи з багатомовними даними.

Переваги та особливості:

- Швидкий та інтуїтивний пошук імен за першою букву
- Паджинація для зручного перегляду великої кількості даних
- Детальна історична інформація про кожне ім'я
- Захищена робота з базою даних через підготовані запити
- Адаптивний дизайн з поддержкою різних пристроїв
- Чистий та легко розширюваний код

Область застосування:

Цей проект може бути використаний як:
- Дослідницький інструмент для аналізу популярності імен
- Довідково-інформаційна система про імена та їх статистику
- Основа для більш складного додатку аналізу демографічних даних
- Навчальний приклад для вивчення PHP та веб-розробки

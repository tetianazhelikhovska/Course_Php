Короткий опис

Це простий PHP-проєкт, який містить одну початкову сторінку `index.php`, мінімальні стилі в `styles/simple.css` та папку `images/` з текстовими файлами.

Функції
- Одна сторінка: `index.php` — точка входу для веб-інтерфейсу.
- Стилі: `styles/simple.css` — базова стилізація для сторінки.
- Ресурси: папка `images/` містить кілька `.txt` файлів (`IMG_0933.txt`, `IMG_3280.txt`, `IMG_4944.txt`, `IMG_9066.txt` та `file.txt`) — текстові представлення зображень.

Тема та особливості
- Тема: мінімалістичний та легкий фронтенд — призначений для швидкого перегляду або прототипу.
- Особливості:
  - Простота структури — легко читати й модифікувати.
  - Без зовнішніх залежностей — підходить для запуску локально з PHP.
  - Файли в `images/` можуть бути місцем для зберігання назв/метаданих зображень або тимчасових даних.

Структура проекту
- `src/`
  - `index.php` — головна сторінка (серверний скрипт PHP).
  - `styles/simple.css` — базові CSS-стилі.
  - `images/` — ресурси/заповнювачі (текстові файли).
